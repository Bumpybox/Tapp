AUTO INTSTALL


-REQUIREMENTS

- Python (http://www.python.org/ftp/python/2.6.4/python-2.6.4.amd64.msi)
- PyQt (http://sourceforge.net/projects/pyqt/files/PyQt4/PyQt-4.10.1/PyQt4-4.10.1-gpl-Py2.7-Qt4.8.4-x64.exe)
- Copy "pythonInstallFolder/Lib/site-packages" into "mayaInstallFolder/Python/Lib/"
- MG_Tools (https://github.com/giordi91/MG_Tools/archive/update1.2.zip) > THIS WILL CHANGE WHEN RELEASING!

-INSTALL

- Download and Install the required software above.
- Download and Extract MG_Tools
- Run 'installGUI.py'
- Enjoy the new awesome MG_Tools menu in Maya:)


##############################################################################

MANUAL INSTALL :

If for any reason you dont have PYQT so you cannot run the UI's you can still use the bundle by installing it manually :

copy the .mll file  for example 

"..\MG_Tools\build\1.2\win\2013x64\pro\MG_rigToolsPro.mll"

into your :

"mayaInstallFolder\bin\plug-ins"

then copy the  python folder inside :

"mayaScriptFolder/MG_Tools"


you need to create the MG_tools  folder.

That s it. Unluckily if you dont have the PYQT you cannot run the custom menu UI but you can still manually call the scripts
from the script editor or manually connect the node , if you want to know more about how do it check the example files or the 
relative doc


I will try in the future to convert the UI to pyside since maya 2014 should have it embedded.
